# Test-About-Future-Job (Client side)

This part was implemented by using _*React*_ library.
The site contains next components:  

1. **Login**: applicants enter their credential before starting the test.
2. **Test**: applicants completing the test according to their studying status.
3. **Results**: applicants get their future job with description and recommendation about enrolling (department).

Next packages were used:

* _*react-router-dom*_: for moving and redirecting between routes;
* _*react-phone-number-input*_: contains styled input for phone numbers with possibility to choose country, where applicants come from.
